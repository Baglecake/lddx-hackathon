What is this?

Our AI diagnostic pipeline generates differential diagnoses for clinical cases. To evaluate accuracy, we compare its output against ground truth diagnoses. The problem: the pipeline might say "Diabetic Polyneuropathy" while the ground truth says "Diabetic peripheral neuropathy" — clinically the same thing, but our evaluator marks it wrong.

This dictionary tells the evaluator which terms are equivalent. Your job: review it, fix errors, and add missing synonyms.

What to do

  • Section 1 (START HERE): Known failures — the pipeline got it right but the evaluator said wrong. Decide if each pair matches.
  • Section 2: Existing synonyms — scan tables, flag wrong mappings, add missing aliases.
  • Section 3: Hierarchies — check parent/child relationships are correct.
✓ Match = Same condition
✗ Not a match = Different condition
? Unsure = Needs discussion
+ term = Write in a missing alias
